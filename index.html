Now Prototyping

Livestock GPS Tracker

A full-stack IoT prototype: ESP32 main collar (GPS + GSM + LoRa) + sub-collar anti-theft mesh (LoRa + strap sensor) + Express API + Supabase/PostgreSQL + Web dashboard + Android app. Built for South African emerging farmers to monitor livestock in real time.

Main Collar

  • ESP32-WROOM-32
  • NEO-6M GPS module
  • SIM800L GSM (cellular)
  • SX1278 LoRa (868 MHz)
  • TP4056 LiPo charger
  • 5000mAh + 1W solar
  • Deep sleep: 5min intervals
  • ~$43/unit (cellular)

Sub-Collar

  • ESP32 + SX1278 LoRa
  • Strap continuity sensor
  • Detects collar cut/opening
  • Battery voltage monitoring
  • Heartbeat every 60 seconds
  • LoRa mesh up to 10 subs
  • 5 min timeout = stray alert

Server & API

  • Express.js + Supabase
  • helmet · cors · rate-limit
  • X-API-Key auth for collars
  • PostGIS geospatial queries
  • Geofence breach detection
  • RLS per-farmer data isolation
  • REST: 7 endpoints

Apps & Dashboard

  • Web dashboard (map-based)
  • Android app (Kotlin)
  • Real-time location view
  • Animal list & status
  • Historical trail playback
  • Battery & health alerts
  • Supabase realtime sync
