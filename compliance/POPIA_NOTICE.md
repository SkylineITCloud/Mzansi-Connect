## 3. Data Subject Rights (Section 5–18)

Under POPIA, data subjects have the right to:
- **Access** — Request a copy of all personal data held
- **Correction** — Rectify inaccurate data
- **Deletion** — Request deletion (subject to legal retention)
- **Objection** — Object to processing for direct marketing
- **Portability** — Receive data in machine-readable format

## 4. Data Retention (Section 14)

- Contact form submissions: retained for 3 years
- Newsletter subscriptions: retained until unsubscribed
- Logs/audit trails: retained for 12 months
- IoT device data: retained for duration of service agreement

## 5. Security Measures (Section 19–22)

- **Encryption**: TLS for data in transit; database at rest
- **Access control**: API key authentication for admin endpoints
- **Rate limiting**: Prevents abuse and brute force
- **Input validation**: All user input sanitized
- **Breach notification**: Notify Information Regulator within 72 hours
